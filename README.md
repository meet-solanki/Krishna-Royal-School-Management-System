# Krishna Science School Management System

## 📌 Project Overview

Krishna Science School Management System is a web based application developed using **ASP.NET** and **SQL Server**. The main objective of this project is to digitize school management operations such as student records, teacher management, attendance, fees and reporting system.

This system reduces manual work and provides an efficient way to manage school data securely.

---

## 🎯 Objectives

• To manage student information digitally
• To manage teacher records
• To maintain attendance records
• To manage fee collection details
• To generate reports easily
• To reduce paperwork

## 🚀 How to Run Project

Step 1:
Install Visual Studio

Step 2:
Install SQL Server and SQL Server Management Studio

Step 3:
Open SQL Server and create database

Step 4:
Import database file from Database folder

Step 5:
Open project in Visual Studio

Step 6:
Change connection string in Web.config file

Step 7:
Run the project
